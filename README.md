## How to use

### Local

- Clone the [local-version branch](#)
- Update node_modules: `npm install`
- To run: `npm start`
- Open the app using [http://localhost:3000/](http://localhost:3000/)

### Online

See the **demo** on [Github page](https://raigyo.github.io/react-chat/)
